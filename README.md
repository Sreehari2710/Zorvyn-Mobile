# Zorvyn Finance

**Zorvyn** is a premium, state-of-the-art personal finance companion built with React Native and Expo. It provides a seamless and visually stunning experience for tracking expenses, managing savings goals, and gaining deep insights into your financial habits.

## ✨ Key Features

### 🏠 Dashboard (Home)
- **Real-time Balance Tracking**: Instant overview of your total balance, monthly income, and expenses.
- **Goal Progress**: Visual tracking of your active savings goal directly on the home screen.
- **Weekly Spending Chart**: Interactive bar chart showing your spending trends over the current week.
- **Recent Activity**: Quick access to your last few transactions with beautiful categorization icons.

### 💰 Transaction Management
- **Full CRUD Operations**: Add, view, edit, and delete transactions with ease.
- **Smart Categorization**: Preset categories (Food, Housing, Salary, Transport, etc.) with unique color-coded icons.
- **Date & Note Picker**: Keep detailed records with customizable dates and descriptive notes.

### 🎯 Savings Goals
- **Monthly Targets**: Set specific savings goals for each month.
- **Intelligent Strategy**: Receive smart tips on how much you can spend daily to stay on track.
- **Streak Tracking**: Stay motivated with daily streaks and best-streak records.
- **Goal History**: Archive and review past goals to see your progress over time.

### 📊 Advanced Insights
- **Category Breakdown**: Interactive donut charts showing where your money goes.
- **Spending Trends**: Analyze your spending over different time periods (This Month, Last Month, 3 Months).
- **Week-over-Week Analytics**: See if you are spending more or less compared to the previous week.
- **Financial Forecasts**: Projected savings based on your current spending habits.

### 👤 Profile & Personalization
- **Theme Engine**: Support for **Dark Mode**, Light Mode, and System defaults with smooth transitions.
- **Onboarding Flow**: Personalized user experience starting from the first launch.
- **Multi-Currency Support**: Choose your preferred currency (INR, USD, EUR, GBP) with localized formatting.
- **Data Export**: Export your entire transaction history to a CSV file for external analysis.

## 🛠 Tech Stack

- **Core**: [React Native](https://reactnative.dev/) & [Expo SDK 54](https://expo.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction/) (File-based routing)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) with Persistence
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) & [Expo Haptics](https://docs.expo.dev/versions/latest/sdk/haptics/)
- **Data Visualization**: [Victory Native](https://formidable.com/open-source/victory-native/) (powered by [Shopify Skia](https://shopify.github.io/react-native-skia/))
- **Icons**: [Lucide React Native](https://lucide.dev/)
- **Validation**: [Zod](https://zod.dev/)
- **Date Handling**: [date-fns](https://date-fns.org/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/)

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npx expo start
   ```
4. **Open on your device**:
   Scan the QR code using the Expo Go app (Android) or the Camera app (iOS).

## 🎨 Design Philosophy

Zorvyn follows a **"Modern Premium"** design language:
- **Glassmorphism**: Subtle transparencies and blurs.
- **Vibrant Gradients**: Carefully curated color palettes for income and expenses.
- **Micro-interactions**: Haptic feedback on buttons and smooth layout transitions.
- **Accessibility**: Clear typography and high-contrast modes for better readability.
